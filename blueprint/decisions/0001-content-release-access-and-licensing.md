# ADR 0001: Public source, licenses, and restricted material

**Status:** accepted
**Updated:** 2026-07-14

The book, course mappings, lab specifications, project briefs, and instructor
templates are public source. Courseware uses CC BY-SA 4.0, software uses MIT, and
hardware-design source uses CERN-OHL-P-2.0 unless a file records a compatible
exception.

Live assessments, current solutions, student records, personal data, credentials,
security-sensitive deployment details, and partner-confidential material are not
stored in this repository. Public files may use opaque identifiers to refer to an
approved separate system, but must not disclose its location or access details.

Generated books and rendered circuit derivatives are release artifacts, not
canonical source. Circuit diagrams are maintained as CircuitikZ code; generated
circuit SVG files are not committed.
