# ESE-SAF-FRM-005 — Event, quarantine and waste record / Fiche événement, quarantaine et déchets

Use one controlled ID and attach protected personal/medical details separately. Report
facts promptly; do not delay reporting to complete every field. / Utilisez un ID maîtrisé
et séparez les données personnelles/médicales protégées. Signalez vite sans attendre tous les champs.

## B. Quarantine tag and chain / Étiquette et chaîne de quarantaine

Attach bilingual `DO NOT USE / NE PAS UTILISER`; do not obscure hazard markings. / Fixez
l'étiquette bilingue sans masquer les dangers.

| Field / Champ | Entry / Valeur |
|---|---|
| Asset/material ID, owner, configuration / ID bien-matière, propriétaire, configuration | |

| Battery chemistry, SoC estimate, Wh, temperature if safely known / Chimie, charge estimée, Wh, température si sûre | |
| Known residual energy and isolation state / Énergie résiduelle connue et isolement | |

Unknown state means restricted access, not zero energy. / Un état inconnu impose l'accès restreint, pas « énergie nulle ».

## C. Investigation and learning / Enquête et apprentissage

Expected versus observed; direct and contributing conditions; controls that worked/
failed; evidence limitations; related incidents/near misses; affected EN/FR assets,
equipment, rooms and cohorts; corrective/preventive actions with owner/date/effectiveness
measure. Avoid blame and unsupported root-cause claims. / Attendu-observé ; conditions ;
protections ; limites ; antécédents ; ressources, matériel, locaux et cohortes touchés ;
actions responsables/dates/mesures d'efficacité. Évitez blâme et cause non prouvée.
