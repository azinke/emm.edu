# ESE-SAF-FRM-003 — Facility inspection protocol and record / Protocole et fiche d'inspection des locaux

**Template only—no inspection represented / Modèle seul—aucune inspection constatée**  
Result vocabulary / Résultats : `PASS / FAIL / NOT TESTED / N/A WITH REASON`.

## Inspection identity / Identité de l'inspection

Institution/site/room; intended L-levels and occupancy; inspection date/time; competent
inspector and authority; applicable rule/edition; prior record; test instrument ID,
function/calibration status; room configuration/photos/evidence IDs. / Établissement,
site/local ; niveaux/effectif ; date/heure ; inspecteur compétent et autorité ; règle/
édition ; fiche antérieure ; instrument et état ; configuration et preuves.

## Method / Méthode

Inspect de-energized where possible; use only competent persons and approved test
methods for energized verification. Do not press an emergency device or discharge an
extinguisher without the authorized test plan. Record an objective result—not a tick
without evidence. A life-safety defect closes the affected room/activity until disposition.

Inspectez hors énergie si possible ; les vérifications sous tension suivent une méthode
approuvée par personne compétente. N'actionnez pas une coupure ni un extincteur sans plan
d'essai autorisé. Consignez un résultat objectif. Un écart vital ferme le local/l'activité concerné.
